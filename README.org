#+TITLE: deprecation-detective
#+AUTHOR: aygp-dr

A babashka CLI tool that scans codebases for deprecated APIs, libraries, and patterns.

** Quick Start

#+begin_src bash
bb run -- --dir ./my-project
bb run -- --dir ./my-project --format json
bb run -- --dir ./my-project --severity high
#+end_src

** Requirements

- [[https://github.com/babashka/babashka][Babashka]] v1.12+

** Usage

#+begin_src
DeprecationDetective — find deprecated code patterns

  -d, --dir DIR        Directory to scan (default: .)
  -f, --format FORMAT  Output format: text, json, edn (default: text)
  -s, --severity LEVEL Minimum severity: low, medium, high (default: low)
  -h, --help           Show help
#+end_src

** Supported Languages

| Language | Patterns | Examples |
|----------|----------|---------|
| Python | 5 | distutils, imp, optparse, cgi, unittest.makeSuite |
| JavaScript | 3 | var usage, arguments object, domain module |
| Java | 1 | java.util.Date constructors |
| Go | 1 | io/ioutil package |

** Exit Codes

- =0= — no deprecations found
- =1= — deprecations found (check output)
- =2= — error (invalid arguments, etc.)

** Output Formats

*Text* (default):
#+begin_src
Found 3 deprecation(s):

  src/main.py:12 [HIGH] distutils is deprecated since Python 3.10, use setuptools
    -> Replace with: setuptools
    |  from distutils.core import setup

Summary: 2 high, 1 medium, 0 low
#+end_src

*JSON* (=--format json=):
#+begin_src json
{
  "total": 3,
  "by-severity": {"high": 2, "medium": 1, "low": 0},
  "findings": [...]
}
#+end_src

** Adding Patterns

Edit =src/deprecation_detective/core.clj= and add entries to the
=deprecation-patterns= vector. Each pattern needs: =:id=, =:lang=,
=:pattern= (regex), =:message=, =:severity=, =:replacement=.
